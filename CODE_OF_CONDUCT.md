# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, collaborate in good faith, and assume the best of others. Maintainers will enforce these standards across all project spaces (issues, PRs, discussions, and any official channel).

## Scope

This applies to every space the project occupies — GitHub repository, issues, pull requests, discussions, and any other communication channel maintained by the project. It also applies when an individual is officially representing the project in public spaces.

## Reporting

If you witness or experience behavior that you believe violates this code, please report it privately to the maintainers at **amarantee.reis@gmail.com**.

Reports will be:

- Reviewed and investigated promptly and fairly
- Kept confidential
- Acted on through whatever response the maintainers judge appropriate (which may include warning, temporary ban, or permanent ban)

Maintainers are obligated to respect the privacy and safety of the reporter.
